# CodeAlpha_Credit-Scoring-Model

## Objective
Predict an individual's creditworthiness using past financial data.

## Approach
- Logistic Regression
- Decision Tree
- Random Forest

## Features
- Income
- Credit Amount
- Loan Duration
- Payment History
- Engineered financial ratios

## Evaluation Metrics
- Precision
- Recall
- F1-Score
- ROC-AUC

## Result
Random Forest achieved the best ROC-AUC and better recall for risky customers.
